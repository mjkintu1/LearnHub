1. Complex Numbers

Definition: A complex number is in the form a + bi, where a and b are real numbers, and i is the imaginary unit with i² = -1.

Example 1: (3 + 2i) + (4 - 5i) = 7 - 3i

Example 2: (1 + 2i)(3 - 4i) = 3 - 4i + 6i - 8i² = 3 + 2i + 8 = 11 + 2i

2. Calculus - Differentiation

Definition: Differentiation finds the rate at which a function changes. The derivative of f(x) gives the slope at any point x.

Example 1: f(x) = x² + 3x + 2 → f'(x) = 2x + 3

Example 2: f(x) = sin(x) → f'(x) = cos(x)

3. Calculus - Integration

Definition: Integration is the reverse of differentiation and helps calculate area under a curve.

Example 1: ∫(2x + 1) dx = x² + x + C

Example 2: ∫sin(x) dx = -cos(x) + C

4. Matrices

Definition: A matrix is a rectangular array of numbers. Operations include addition, multiplication, and finding determinants.

Example: A = [1 2; 3 4], B = [2 0; 1 3]

A × B: [1×2 + 2×1, 1×0 + 2×3; 3×2 + 4×1, 3×0 + 4×3] = [4, 6; 10, 12]

5. Vectors

Definition: A vector represents magnitude and direction. Common operations include dot product and cross product.

Example 1: Magnitude of v = (3, 4) is |v| = √(3² + 4²) = 5

Example 2: Dot product: a = (1, 2), b = (3, 4) → a·b = 1×3 + 2×4 = 11

6. Probability

Definition: Probability is the measure of the likelihood of an event occurring, between 0 and 1.

Example 1: P(rolling a 6) = 1/6

Example 2: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

7. Logarithms

Definition: A logarithm is the inverse of exponentiation. logb(x) = y means by = x

Example 1: log10(100) = 2

Example 2: log2(8) = 3

8. Sequences and Series

Definition: A sequence is a list of numbers. A series is the sum of a sequence.

Example: Arithmetic sequence: 2, 5, 8, 11 → Common difference = 3

Sum of first n terms: Sn = n/2 × (2a + (n - 1)d)

9. Limits

Definition: A limit describes the value that a function approaches as the input approaches a certain value.

Example: limx→0 (sin x / x) = 1
